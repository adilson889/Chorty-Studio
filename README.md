# Chorty Studio

Chorty Studio é um ambiente inovador para criar sites e aplicações web, usando comandos simples e totalmente em português. Com Chorty Studio, você elimina as barreiras do inglês e da complexidade dos códigos tradicionais, tornando o desenvolvimento web acessível para todos.

## Objetivo

O principal objetivo do Chorty Studio é democratizar o desenvolvimento web, permitindo que pessoas com pouca ou nenhuma experiência em programação criem projetos reais, utilizando apenas a língua portuguesa. Este projeto facilita a entrada de novos desenvolvedores no universo da tecnologia, além de ser uma ferramenta poderosa para educadores.

## Finalidades

- Tornar a programação intuitiva para iniciantes e não programadores.
- Fornecer uma plataforma de aprendizado em português, focada na construção prática de sites.
- Servir como recurso didático para escolas, professores e cursos.
- Promover a inclusão tecnológica de comunidades de língua portuguesa.
- Acelerar a criação de protótipos e ideias web de forma descomplicada.

## Principais funcionalidades

- Comandos como “criar pagina”, “adicionar botao” e “definir variavel” são convertidos automaticamente em código HTML, CSS e JavaScript funcional.
- Interface amigável, desenvolvida pensando em quem está começando.
- Estrutura flexível tanto para aprendizado quanto para projetos rápidos.

## Como usar

1. Faça o clone do repositório:
   ```
   git clone https://github.com/adilson889/Chorty-Studio
   ```
2. Abra o arquivo index.html no navegador de sua preferência.
3. Utilize a interface para escrever comandos em português e veja seu site tomando forma automaticamente.

## Exemplo prático

Comandos como:
```
criar pagina "meu-site"
adicionar botao "Clique aqui"
definir variavel "nome" = "Seu nome"
```
São interpretados instantaneamente e geram um site funcional, pronto para ser customizado ou publicado.
